### What I learned

This was my first Frontend Mentor challenge. I learned how to read design values directly from Figma — spacing, border-radius, colors, and typography. I also practiced BEM naming convention for the first time, which made my CSS more organized and readable.

Centering elements with Flexbox:
```css
body {
    display: flex;
    align-items: center;
    justify-content: center;
    min-height: 100vh;
}
```

I also learned that `box-sizing: border-box` prevents unexpected layout issues, and that placing HTML elements outside `<body>` causes structural bugs.
